# Uniapp-vue2-starter

Uniapp 模板，使用 vue2 生态

特性：

- uview2
- vuex
- unocss

## Project setup

```
pnpm install
```

### Compiles and hot-reloads for development

```
pnpm run serve
```

### Compiles and minifies for production

```
pnpm run build
```
